# Toronto-road-collision-severity-analysis
This project analyzes road collision patterns, severity levels, and key risk factors across Toronto using open data from the Toronto Police Service. The goal is to identify where, when, and why severe collisions occur, and provide data-driven insights to support traffic safety improvements and policy decisions.


Dataset Link : https://data.torontopolice.on.ca/datasets/TorontoPS::traffic-collisions-open-data-asr-t-tbl-001/explore
